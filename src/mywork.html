GWC

This project was the exam project for 2nd semester where we had to collaborate with a startup. We collaborated with a startup called Green Wardrobe Club, which is a startup that provides a marketplace where you can lend and rent out exclusive clothings that you only wear rarely. In this project we had to redesign their webpage, create a social media strategy and video content for social media. My main role in this project was video and design.

  2. Premiere Pro editing
  3. After Effects
  4. HTML, CSS, JS

Musik
loftet

This was a project we did for Musikloftet. Musikloftet rents out rehearsal rooms for small and larger bands. The goal was to make it simple for the user to navigate and to find information about the different rooms that was suitable for their needs. We chose to focus on the segment of bands that perform on medium-sized stages and small festivals. Our keywords for the design was musicality, creativity and community. The result was a minimalistic look with a fifth element dominating the design.

Skills and software

  1. User research
  2. Figma prototyping
  3. Photoshop/Illustrator
  4. HTML, CSS Tailwind
  5. User testing

Skole
Mad

This was a project we did for Skolemad, a Danish system where you can manage and order food for your children at school. They have a lot of users, because parents are being forced to use it. However, the website has a lot of information on it, and the user experience is not good. It is very difficult to scan the content because it is not providing the user any help in terms of white space and contrast between headings and paragraphs. In essence, in this project the main focus was to optimize the user experience, reduce the amount of clicks and to make the user flow intuitive.

Skills and software

  1. User research
  2. Figma prototyping
  3. HTML, CSS Tailwind

Vild
skud

Vildskud Festival is a festival for performance art in Copenhagen. This was a redesign project with the focus of creating sustainable web design. Each page was limited to not exceed more than 0,5 MB. This constraint challenged us to experiment more with colors an typography rather than big images. The hero background and logo was designed by me using Photoshop and Illustrator.

Skills and software

  1. Sustainable webdesign
  2. Redesign
  3. Figma prototyping
  4. HTML, CSS
